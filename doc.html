Rotas

  • /api

    • Método: GET

    200 OK.
    Retorno: Objeto JSON caso apenas um registro novo, array de objetos JSON caso dois ou mais.

    204 No Content.
    Retorno: Array vazio.

    Compara os registros de caixas fechados no banco de dados com os arquivos JSON no diretório /files.
    Caso não encontre o arquivo do caixa, consulta o banco de dados, escreve o arquivo no diretório e o retorna na requisição.
    Caso não exista novos registros para escrever, retorna um array vazio.

  • /api/helper.php

    • Método: GET

    200 OK.
    Retorno: Objeto JSON contendo id e nome das formas de pagamento e categorias de produtos.

    Consulta o banco de dados e retorna as formas de pagamento e categorias de produtos.
    Útil verificar quando receber algum id desconhecido na requisição da rota /api,
    o usuário pode ter cadastrado alguma forma de pagamento ou categoria de produtos nova.
    Escreve o arquivo "helper.json" no diretório /files.

  • /api/files

    • Diretório contendo os arquivos JSON de todos os caixas, e o arquivo "helper.json".
